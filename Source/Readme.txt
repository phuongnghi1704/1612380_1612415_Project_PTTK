Cách build source code:
Bước 1: Vào trang web này để install Node https://nodejs.org/en/download/
Bước 2: Vào folder chứa source của app (admin-website hoặc customer-website)
Bước 3: Mở terminal tại folder đó
Bước 4: Gõ "npm install" để install các thư viện cần thiết
Bước 5: Gõ "npm start" để chạy app trên localhost
Bước 6: Dùng trình duyệt vào địa chỉ sau đây để truy cập app: http://localhost:3000/

Nếu không thể build source code trên local, thầy có thể xem app bọn em đã deploy lên host heroku
